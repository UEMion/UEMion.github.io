UWP笔记-跨页面访问控件

刚刚看到一个东西很有用，在自己的博客记录一下哈哈
说的是开发UWP时，经常需要在这个页面访问另一个页面的控件
直接MainPage.myFrame直接这样访问当然是不行的，因为myFrame这个控件不是Public，这样即可：
先在XAML界面这个控件myFrame里加上这样一句话

x:FieldModifier=" Public"

然后对应的C#代码里面这样：

    public sealed partial class MainPage : Page
    {

        public static MainPage firstPage;

        public MainPage()
        {
            this.InitializeComponent();
            firstPage= this;
        }

这样之后就可以在别的页面的代码里通过MainPage.firstPage.myFrame访问这个控件了
